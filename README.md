# BatiFlow

Outil de trésorerie prévisionnelle pour artisans BTP solo en France.

BatiFlow aide les plombiers, électriciens, maçons, peintres et menuisiers à visualiser leur trésorerie sur 90 jours, d'un coup d'œil, depuis leur téléphone.

## Stack technique

- Next.js (App Router), TypeScript, Tailwind CSS
- PostgreSQL (Neon)
- Déploiement : Vercel (auto-deploy depuis `main`)

## Développement local

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans le navigateur.
