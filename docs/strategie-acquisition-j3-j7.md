## Conseils d'exécution

### Ce qui marche avec les artisans BTP
1. **Parler leur langue :** pas de jargon tech ou financier. "Savoir si le mois va passer" > "trésorerie prévisionnelle"
2. **Authenticité :** pas de comm corporate. Parler comme un fondateur qui résout un vrai problème
3. **Questions ouvertes :** les posts qui demandent "ça vous est déjà arrivé ?" génèrent beaucoup plus d'engagement que les pitchs directs
4. **Répondre à TOUS les commentaires :** les artisans testent si vous êtes réel avant de s'inscrire
5. **Pas de lien direct dans le post Facebook :** mettre le lien en commentaire (l'algorithme pénalise les posts avec liens externes)

### Ce qu'il faut éviter
- Les messages trop longs (surtout WhatsApp : max 5 lignes)
- Le pitch produit d'emblée (commencer par la douleur)
- Les posts trop "publicitaires" (bannissement groupes Facebook)
- Envoyer le même message mot-à-mot partout (risque bannissement)

### Indicateurs d'alerte
- Taux de réponse WhatsApp < 5 % → changer l'angle du message
- 0 commentaire sur un post Facebook après 2h → le groupe est mort ou le post est bloqué
- 0 réponse DM Instagram après 20 messages → vérifier si compte signalé
